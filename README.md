It contains two Python-based management systems developed to demonstrate core programming concepts, object-oriented programming, file handling, and data management techniques.

1. Hospital Management System

A console-based application designed to manage patient records, doctor information, and appointment scheduling. Patient records are stored using dictionaries, patient IDs are maintained through lists, and classes are used to represent patients and doctors. The system also stores records in files and uses exception handling to validate patient information.

Features

Add and manage patient records
Maintain patient IDs using lists
Store patient details using dictionaries
Patient and Doctor classes using OOP
Schedule appointments
Save records to files
Exception handling for invalid data


2. Exam Result Processing System

A Python application that manages student records and processes examination results. Student details are stored using tuples, marks are maintained in dictionaries, and records are managed through lists. The system calculates totals, percentages, grades, generates rank lists, and saves reports to files.

Features

- Add student records
- Store student details using tuples
- Store marks using dictionaries
- Generate grade reports
- Calculate totals and percentages
- Generate rank lists
- Save results to files
- Exception handling for input validation
- Concepts Demonstrated
- Python Programming
- Object-Oriented Programming (OOP)
- Functions and Modular Design
- Lists, Tuples, and Dictionaries
- File Handling
- Exception Handling
